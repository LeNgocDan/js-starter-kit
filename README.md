# Lucky Draw

1. **Clone this repository.**
- `git clone https://github.com/ngcdan/lucky-draw.git`
3. **Install Node Packages.**
- `npm install`
4. **Run the app.**
- `npm start`

Access webui via localhost:3000
Access rest api via localhost:3001
